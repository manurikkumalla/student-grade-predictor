# Student Grade Predictor

An academic planning application built to calculate GPAs, track course credits, and mathematically predict grades needed to hit a target CGPA.

## Features
- **GPA Calculation**: Real-time weighted average computation based on credits.
- **Predictive Analytics**: Calculates the exact grade average required on remaining hours to reach a user-defined target.
- **Color-Coded Feedback**: Visually flags target plausibility (green, blue, red).
